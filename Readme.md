# 🥭 Aamrai - Mango Website

Aamrai is a simple and responsive multi-page website created using **HTML** and **CSS**. The website showcases different varieties of mangoes, information about the orchard, and a contact form for visitors.

## 📌 Project Overview

This project was developed to demonstrate fundamental web development concepts such as:

- HTML page structure
- CSS styling
- Navigation between multiple pages
- Image integration
- Hover effects
- Flexbox layout
- Forms
- Tables
- Lists

## 🚀 Features

### 🏠 Home Page
- Welcome section
- Information about Aamrai
- Mango varieties gallery
- Special features section
- Seasonal availability table
- Responsive card layout using Flexbox

### 📖 About Page
- Introduction to Aamrai orchard
- Mission statement
- Types of mangoes available
- Benefits of mangoes
- Reasons to choose Aamrai

### 📞 Contact Page
- User-friendly contact form
- Name input
- Email input
- Message textarea
- Submit button

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
